# Enhanced Text-to-Speech Web Application

A simple web-based text-to-speech converter built using Python, Flask, and gTTS, with a modern UI using Bootstrap.

## Features

- Convert text to speech in multiple languages.
- Download the audio file after conversion.
- Choose between normal or slow speech.
- Clean, responsive UI with error handling.
